## SPAとしてのビルド手順(yarnの場合)

モジュールのインストール:

```bash
yarn
```

ビルド:

```bash
yarn build
```

buildフォルダの中身をサーバー等に配置し、index.htmlにアクセスすることでアプリを実行できる。
